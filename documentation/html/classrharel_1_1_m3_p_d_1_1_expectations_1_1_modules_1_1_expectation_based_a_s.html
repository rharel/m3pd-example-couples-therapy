Detailed Description

An implementation of Agency.Modules.ASModule that uses expectation arrangements to derive action.

A list of candidate moves is derived from an interaction expectation arrangement, and then a user-supplied method selects the winning candidate.

This module requires the information state to support the following components:

  1. Type: SocialContext Identifier: CommonComponentIDs.SOCIAL_CONTEXT
