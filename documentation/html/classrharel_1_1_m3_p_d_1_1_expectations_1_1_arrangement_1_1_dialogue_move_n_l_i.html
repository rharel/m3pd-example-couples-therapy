Detailed Description

Helps make move composition easier via an interface that is closer to natural language descriptions of dialogue moves.

using static rharel.Expectations.Arrangement.DialogueMoveNLI;
struct Greeting { ... }
...
Move("greeting",
addressee: "Alice",
properties: new Greeting(...)
);

NLI stands for Natural Language Interface.
