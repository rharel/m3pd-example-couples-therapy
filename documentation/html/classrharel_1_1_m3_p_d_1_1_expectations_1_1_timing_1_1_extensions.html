Evaluates an enumeration of rules for the specified agent at this time.

Template Parameters
TThe type of the implication the rules derive.
Parameters
rulesThe enumerable to evaluate.
idThe identifier of the agent against which the rules should be evaluated.
Returns
The implication supported most heavily, or None if no rules were relevant for id .

The rules each vote for their own implication with their weight. The implication with the highest total weight is the one that will be returned. In case of a tie, the rule appearing last in the enumeration wins.

ArgumentNullExceptionWhen id is null.
