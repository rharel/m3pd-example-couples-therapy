Detailed Description

A helper class for the generation of a combined hash code made up of multiple components.

int hash = HashCombiner.Initialize();
hash = HashCombiner.Hash(hash, "Hello world!");
hash = HashCombiner.Hash(hash, 2.5f);

return hash;

Configures the combiner's parameters.

Parameters
seedThe initial seed.
factorThe multiplication factor.

The combination algorithm used is dependent on two parameters: an initial seed and a multiplication factor. The seed is just a value yielded when calling Initialize. The factor is being used to combine two existing hash-codes into one:

int first_hash = seed;
int combined_hash = first_hash * factor + second_hash;
