Detailed Description

Helps make program composition easier via an interface that is closer to natural language descriptions of expectation arrangements.

var expect = new NodeNLI();
expect.Sequence("some sequence",
expect.Event(...),
expect.Conjunction(...),
...
);

If during the creation of a new node no identifier is supplied, one is generated automatically. Identifiers prefixed with an _underscore should be considered reserved for this purpose.

NLI stands for Natural Language Interface.
